## ðŸš€ Building from Source

```bash
git clone https://github.com/Mylifeismyhome/Websocket.git
cd Websocket
mkdir build
cd build
cmake -DCMAKE_CXX_FLAGS="-m64" \
      -DENABLE_C_API=ON \
      -DENABLE_CPP_API=ON \
      -DBUILD_SHARED=ON \
      -DBUILD_STATIC=ON \
      -DEXAMPLE_C_API=ON \
      ../
make
```

This will build both static and shared libraries, along with optional examples if enabled.
